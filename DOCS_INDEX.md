## 🗂️ Cách sử dụng docs này

### Nếu bạn là Developer mới:
1. Đọc **README.md** trước
2. Đọc **FEATURES_COMPLETE.md**
3. Setup local theo README
4. Đọc **CONTRIBUTING.md** nếu muốn contribute

### Nếu bạn muốn Deploy:
1. Đọc **START_HERE.md** ⭐
2. Làm theo **QUICK_DEPLOY.md** ⭐⭐⭐
3. Check **PRE_DEPLOY_CHECKLIST.md**
4. Deploy!

### Nếu bạn gặp lỗi:
1. Check **FIXES_SUMMARY.md**
2. Tìm file fix tương ứng
3. Đọc **FIREBASE_DEBUG.md** nếu lỗi Firebase
4. Google error message

### Nếu bạn muốn hiểu feature:
1. Đọc **FEATURES_COMPLETE.md** overview
2. Đọc file feature cụ thể (MESSENGER_FEATURE.md, etc.)
3. Xem code trong `client/src/components/`
