# Public Assets

## Logo

Đặt file logo của bạn vào đây với tên `logo.jpg` hoặc `logo.png`

Kích thước đề xuất: 200x200px hoặc lớn hơn (sẽ tự động scale về 40x40px)

Nếu không có logo, bạn có thể:
1. Tạo logo tại: https://www.canva.com/
2. Hoặc sử dụng icon/emoji làm logo tạm
3. Hoặc để trống, chỉ hiển thị text "Code Net"
